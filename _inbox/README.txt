CHARAMARL 素材受け取りフォルダ
==============================
ここに素材を入れて「入れた」と伝えてください。取り込みます。

色キー: red / yellow / green / cyan / blue / pink

[1] illustrations/  … イラスト
    パターンA（6色を手調整済み）: <id>_<色>.png を6枚
        例) sue_red.png, sue_yellow.png, ... sue_pink.png
    パターンB（ベース1枚から自動生成）: <id>.png または <id>.ai を1枚
        例) sue.png  → スクリプトで6色生成します
    ※ id は sue / putti / mossun

[2] products/       … 実物写真（色別・任意）
    <id>_<色>.png   例) sue_red.png ... (白背景でOK)

[3] other/          … バナー画像・追加キャラ・その他なんでも
